Plugin QGIS Open ICGC

Un plugin QGIS para acceder a los datos abiertos publicados por el ICGC (Institut Cartogràfic de Catalunya).

Permite realizar búsquedas de topónimos, calles, carreteras y coordenadas en diferentes sistemas de referencia:

cargar capas base de información:

gestionar series temporales:

búsqueda de fotogramas históricos raster:

descargar información vectorial o ráster por área, municipio o comarca (dependiendo del producto):

hacer una gestión básica del estilo de visualización:

y mostrar ayuda:

Se añadiran más datos y servicios próximamente.

Este plugin usa las librerías suds-py3, wsse y los geoservicios del catastro
